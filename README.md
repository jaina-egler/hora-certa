# HoraCerta IFRO

Sistema web para orientação e simulação de Atividades Complementares dos cursos de graduação do Instituto Federal de Rondônia.

## Sobre o projeto

O HoraCerta IFRO é uma aplicação desenvolvida com Flask, com o objetivo de auxiliar estudantes na compreensão das regras de aproveitamento das Atividades Complementares.

A ferramenta permite orientar o estudante sobre tipos de atividades, limites por categoria, formas de comprovação e erros comuns no processo de solicitação.

## Base normativa

O projeto utiliza como referência a Resolução nº 24/REIT-CONSUP/IFRO, de 01 de agosto de 2022, que regulamenta as Atividades Complementares dos cursos de graduação do IFRO.

## Objetivos

- Explicar o que são Atividades Complementares.
- Orientar sobre atividades que podem ser aceitas.
- Apresentar cuidados e erros comuns no lançamento.
- Simular o aproveitamento de carga horária com base nas regras institucionais.
- Receber sugestões, dúvidas e possíveis atualizações da tabela.
