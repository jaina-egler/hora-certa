# Relatório Técnico — HoraCerta IFRO

## 1. Título do projeto

HoraCerta IFRO — Sistema Web para Orientação e Simulação de Atividades Complementares.

## 2. Introdução

As Atividades Complementares fazem parte da formação acadêmica dos estudantes de graduação e permitem o aproveitamento de experiências de ensino, pesquisa, extensão, participação acadêmica, produção técnica, atividades sociais e outras ações formativas.

Apesar de sua importância, muitos estudantes encontram dificuldades para compreender os limites de carga horária, os documentos exigidos e a classificação correta das atividades.

## 3. Problema

A interpretação incorreta das regras pode levar o estudante a acreditar que a carga horária total de um certificado será integralmente aproveitada, sem considerar os limites definidos por categoria.

## 4. Objetivo geral

Desenvolver uma aplicação web para auxiliar estudantes na compreensão e simulação das Atividades Complementares, com base na regulamentação institucional do IFRO.

## 5. Objetivos específicos

- Apresentar informações sobre Atividades Complementares.
- Explicar atividades que podem ser aceitas.
- Apontar erros comuns no processo de lançamento.
- Simular o aproveitamento estimado de carga horária.
- Disponibilizar um canal para dúvidas e sugestões.

## 6. Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript
- Bootstrap

## 7. Versão inicial

A primeira versão do sistema contempla a estrutura Flask, página única informativa, seções de orientação, formulário de contato visual e área inicial do simulador.
